**To Install Ngspice in Linux**

1. Go to terminal
2. sudo apt install ngspice


**To Run**
1. Save the project file with **.sp** extension
2. Go to terminal and go to the location where your project file is saved
3. type the below command
4. ***ngspice -b <file_name>***

**OR**

1. **ngspice**
2. **source <file_name>**
3. **run**
4. **display**                 //displays all the signals in the design , i.e. all nets
5. **print <signal_name>**    //prints the value of the signal
6. **plot <signal_name>**    //plots a graph of the mentioned signal
